Instructions for Running Python Files

This repository contains Python code files in the code folder to solve various questions. Each file is named after the question it solves, such as 1c.ipynb, 2ab.ipynb, etc.

Running the Code
	-To run the code, follow these steps:
	-Clone this repository to your local machine.
	-Navigate to the code folder.

-Execute the Python code sequentially:
1c.ipynb
2ab.ipynb
3ab.ipynb
4a.ipynb
4c.ipynb
5ab.ipynb

Additional Notes
The Python files contain all the relevant functions and descriptions within themselves. There are no external common files being imported.

Other files in the code folder are used by the code files for reading or execution purposes, such as executables or outtags.
